# RoadKeeper Application
위험 요소 정보 전달과 위험 상황 예방을 위한 차량용 인포테인먼트

## 주요 기능
- 실시간으로 도로 영상을 서버에 전송
- 사용자에게 인식된 위험물 데이터 제공
